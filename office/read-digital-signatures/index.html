        Riêng tư tuyệt đối

        File không rời khỏi thiết bị. Không được đẩy lên máy chủ từ xa hay lưu trữ trên đám mây.

        Cách sử dụng

        1

        Chọn file PDF hoặc XML

        Kéo thả hoặc bấm "Chọn file" để thêm một hoặc nhiều file. Có thể thêm lẫn lộn PDF và XML.

        2

        Tự động phân tích

        Ngay khi thêm file, công cụ tự động phát hiện định dạng, đọc chữ ký và giải mã chứng thư số.

        3

        Xem chi tiết

        Chọn file ở panel trái, bấm vào từng chữ ký ở panel phải để mở rộng và xem đầy đủ certificate chain.

        Câu hỏi thường gặp

        Vì sao trạng thái XML thường là "Không xác định"?

        Để xác minh tuyệt đối chữ ký cần thực hiện nhiều bước phức tạp. Công cụ tập trung vào việc đọc thông tin người ký, tổ chức cấp và hiệu lực chứng thư thay vì xác minh chữ ký đầy đủ.

        Hoá đơn điện tử có chữ ký ở PDF hay XML?

        Với hoá đơn điện tử Việt Nam, chữ ký số hợp pháp nằm trong XML (theo chuẩn của Tổng cục Thuế). File PDF là bản hiển thị cho người đọc, thường không có chữ ký số.

        File có được upload lên server không?

        Không. Tất cả phân tích diễn ra trên trình duyệt, file không bao giờ rời khỏi thiết bị.

        Giới hạn kích thước?

        Tối đa 100 MB mỗi file, tối đa 20 file một lần.
