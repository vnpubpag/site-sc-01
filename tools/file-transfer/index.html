Mã hóa đầu cuối

File được truyền trực tiếp giữa hai trình duyệt qua WebRTC. Không có dữ liệu nào được lưu trên server.

Giới thiệu công cụ

Truyền file trực tiếp giữa hai trình duyệt bằng công nghệ WebRTC ngang hàng. Kết nối ưu tiên P2P trực tiếp và tự động chuyển sang TURN relay khi cần. Không giới hạn kích thước file từ phía công cụ.

Riêng tư & Bảo mật

Dữ liệu truyền trực tiếp giữa các trình duyệt. Không server nào lưu trữ file của bạn.

Truyền tải nhanh

Kết nối P2P mang lại tốc độ tối đa, đặc biệt khi cùng mạng LAN.

Hoạt động mọi nơi

Kết nối xuyên mạng với khả năng tự động chuyển sang TURN relay khi cần.

Cách sử dụng

1

Tạo phòng

Nhấn "Tạo phòng" và chia sẻ mã phòng 4 chữ số cho người nhận.

2

Vào phòng

Người nhận nhập mã phòng và nhấn "Vào phòng" để kết nối.

3

Gửi & tải về

Chọn file và nhấn "Gửi file". Người nhận có thể tải về khi hoàn tất.

Câu hỏi thường gặp

File của tôi có được tải lên server không?

Không. File được gửi trực tiếp từ trình duyệt này sang trình duyệt khác qua WebRTC DataChannel. Server tín hiệu chỉ hỗ trợ thiết lập kết nối, không bao giờ nhìn thấy dữ liệu file của bạn.

Kích thước file tối đa là bao nhiêu?

Công cụ không giới hạn kích thước. Tuy nhiên, file quá lớn có thể bị ảnh hưởng bởi bộ nhớ trình duyệt hoặc độ ổn định kết nối. File dưới 1 GB hoạt động ổn định.

Có hoạt động xuyên mạng khác nhau không?

Có. Công cụ sẽ thử kết nối P2P trực tiếp trước. Nếu thất bại (ví dụ: sau NAT nghiêm ngặt), sẽ tự động sử dụng TURN relay server trên Cloudflare.

Hỗ trợ những trình duyệt nào?

Tất cả trình duyệt hiện đại hỗ trợ WebRTC: Chrome, Firefox, Edge, Safari. Cả máy tính và điện thoại đều được hỗ trợ.
