Câu hỏi thường gặp

File PDF có được upload lên server không?

Không. Toàn bộ quá trình chuyển đổi diễn ra trên trình duyệt của bạn. File PDF không bao giờ rời khỏi thiết bị.

Công cụ hỗ trợ loại PDF nào?

Hỗ trợ PDF có nội dung văn bản (text-based) như báo cáo, hợp đồng, hóa đơn. Không hỗ trợ PDF scan (cần OCR), bố cục phức tạp (đa cột, tạp chí) hoặc biểu mẫu.

Chất lượng chuyển đổi như thế nào?

Phù hợp với tài liệu đơn giản. Công cụ tự động nhận diện tiêu đề, đoạn văn dựa trên kích thước chữ và khoảng cách dòng. Bố cục phức tạp có thể không chính xác 100%.

Có giới hạn dung lượng file không?

Công cụ hỗ trợ file PDF tối đa 50 MB và không quá 50 trang. Vì xử lý trên trình duyệt nên file quá lớn có thể chậm trên thiết bị cấu hình thấp.
