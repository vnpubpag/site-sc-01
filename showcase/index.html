06. Overlay

Modal dialog và focus surface

Modal là core component. Cần cân bằng shadow, border, backdrop, hierarchy giữa title, body, footer actions.

Kích hoạt modal

Click button để preview modal với đầy đủ header, body, footer. Kiểm tra backdrop blur, shadow và button grouping.

Danh sách kiểm tra component
  • Button và input đã đi qua component shared.
  • Theme switch chỉ đổi token trên root html.
  • Spacing review: 4 / 8 / 12 / 16 / 24 / 32.
  • Footer và card kiểm tra consistency giữa section.
  • Modal cần test: backdrop, border, shadow, footer bg.
